Detailed Description

HID keyboard helper function implementations.

Author
Jeff Rowberg
Date
2014-11-07

This file provides supporting code for sending HID keyboard data to a connected wired or wireless host. This is only relevant if you have enabled normal HID support over USB or Bluetooth.

Normally it is not necessary to edit this file.
