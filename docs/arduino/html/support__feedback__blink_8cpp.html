Detailed Description

Feedback implementations for single LED.

Author
Jeff Rowberg
Date
2014-11-07

This file defines the feeback functionality for the blinking LED that is built into many microcontroller boards, e.g. the Teensy++. This file is meant to contain only behavioral logic and abstracted functions that do not require specific pin knowledge of the selected MCU. Any pins used should be defined in the selected MCU board support file.

Normally it is not necessary to edit this file.
