enum bluetooth_mode_t

List of possible values for blink mode.

Enumerator
KG_BLUETOOTH_MODE_DISABLED 

Bluetooth is disabled, undiscoverable and unconnectable (radio off)

KG_BLUETOOTH_MODE_AUTOCALL 

Bluetooth enabled and in autocall mode if any pairing entries exist.

KG_BLUETOOTH_MODE_MANUAL 

Bluetooth enabled but will not autocall, and will be visible only if no pairings exist.

KG_BLUETOOTH_MODE_VISIBLE 

Bluetooth enabled and will not autocall, but will be visible in inquiries at all times.
