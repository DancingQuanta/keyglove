void setup_motion_mpu6050_hand ( )

Initialize MPU-6050 communications and interrupt handler.

This function sets the MPU-6050 to 100Hz output, 2000 deg/sec resolution for the gyroscope, and enables a latching active-low interrupt on DRDY (raw data ready), MOT (motion detected), and ZMOT (zero motion detected).

void update_motion_mpu6050_hand ( )

Update motion data from MPU-6050.

This function is called whenever the motion interrupt occurs and the flag is set by the interrupt handler, and then detected as such from inside the main loop() function. All six axes of raw data are read, then filtered and stored.
