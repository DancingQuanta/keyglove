Detailed Description

HID mouse helper function declarations.

Author
Jeff Rowberg
Date
2014-11-07

This file provides supporting code for translating motion into mouse movement data, and for sending this HID mouse data to a connected wired or wireless host. This is only relevant if you have enabled normal HID support over USB or Bluetooth.

Normally it is not necessary to edit this file.
