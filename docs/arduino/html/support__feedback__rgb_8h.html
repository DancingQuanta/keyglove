Detailed Description

Feedback declarations for RGB LED.

Author
Jeff Rowberg
Date
2014-08-09

This file defines the feeback functionality for the RGB LED that may be included as an optional module in a Keyglove modular hardware design. This file is meant to contain only behavioral logic and abstracted functions that do not require specific pin knowledge of the selected MCU. Any pins used should be defined in the selected MCU board support file.

Normally it is not necessary to edit this file.
